# Getting started


## Hello world is one liner
```Scala
"Hello world" echo
```

`niva run main.niva`  
Instead of `echo("Hello world")` we use `"Hello world" echo`


>To run the code use `niva run file.niva` or just `niva run` if file named main  
It may take longer the first time
